## How to Capture Screenshots

### Desktop Screenshots (Chrome DevTools)
1. Open the app in Chrome
2. Press F12 to open DevTools
3. Click the device toolbar icon (Ctrl+Shift+M)
4. Set viewport size to 1920x1080
5. Press Ctrl+Shift+P, type "screenshot", choose "Capture screenshot"

### Mobile Screenshots
1. In DevTools device mode
2. Select "iPhone 12 Pro" or similar
3. Capture screenshot as above

### Creating Demo GIF
Use one of these tools:
- **ScreenToGif** (Windows) - https://www.screentogif.com/
- **LICEcap** (Mac/Windows) - https://www.cockos.com/licecap/
- **Kap** (Mac) - https://getkap.co/

**Recording Settings:**
- Resolution: 1280x720
- Frame rate: 15 FPS
- Duration: 20-30 seconds
- File size: <5 MB
